# Identity

## To preview this site offline:
```
git clone https://github.com/tsondt.github.io
```
* with `ruby` and `bundle`:
```
BUNDLE_PATH=.bundle bundle check || \
bundle install && \
bundle exec jekyll serve
```
* with Docker:
```
make run
```
